About Minecraft Hub

Fan-made community portal for all Minecraft players

Our Mission

Minecraft Hub was created by long-time Minecraft players to unite builders, redstone engineers, modders, server owners and casual fans on one simple website.

We showcase outstanding player builds, share mod installation guides, highlight shaders and texture packs, and help users find active Java & Bedrock Edition community servers.

This project is fully fan-run and not affiliated with Mojang Studios or Microsoft. Our goal is to celebrate the creativity that makes Minecraft one of the best-selling games of all time.

Project Background

This website was built as a CS50x final project using pure HTML, external CSS and vanilla JavaScript, demonstrating responsive web design and client-side interactivity.
