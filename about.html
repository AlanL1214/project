About Minecraft Hub

Fan made community for all Minecraft players

Our Mission

Minecraft Hub was created by me, a guy who's doing cs50, for 1 single purpose, to create a beginner friendly community to help new players. (its for cs50 but still counts).

We showcase outstanding player builds, help players install mods, and help users find active minecraft java servers.

This project is fully ran by a fun and not affiliated with Mojang Studios or Microsoft. Our goal is to create a community to help introduce and help new players to minecraft.

Project Background

This website was built for a cs50 final project using HTML, CSS and javascript, making a web design and client-side interactivity.
